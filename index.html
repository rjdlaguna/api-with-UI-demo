The following are the available routes

Root Route: Displays available routes.

GET /api/users: Returns all users.

GET /api/users/1: Fetches a specific user by ID.

GET api/users/8/Valdez/a@email.com/20/12345: Retrieves and displays parameters.

POST /api/users: Adds a new user through form submission.

DELETE /api/delete/1: Deletes a user by ID.
